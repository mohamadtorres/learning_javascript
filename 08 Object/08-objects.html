Object

An Object groups multiple values together
Objects are created using curly braces {}
Objects can contain properties and methods
Properties are named data values that can be accessed using dot notation or bracket notation
Methods are functions that can be called on objects
we can change the properties of an object using dot notation or bracket notation ===> product.price = 1500

Two different ways to reach an element in an object

console.log(product2.name) // to access a property in an object
console.log(product2['name']); // to access a property by Brqacket Notation

Why we use bracket Notation?

Sometimes for example when u have:

console.log(product2.delivery-time)

it does not understand that the property is talking about an attribute called delivery time. it sees the "-" as a minus and it wants to subtract the time from Delivery which is wrong because you dont want to do it. So you put it in brackets to show this is a property.

console.log(product2['delivery-time']) // will work correctly
